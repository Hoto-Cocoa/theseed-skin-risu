# risu
the seed engine 및 호환 엔진용 밝고 둥근 파스텔 핑크 스킨입니다.

## 위키 설정
| 키 | 설명 | 기본값 |
|---|---|---|
| `skin.risu.logo_text` | 내비게이션 바 로고 텍스트 | `wiki.site_name` |
| `skin.risu.logo_image` | 로고 이미지 URL (설정 시 텍스트 대신 표시) | 없음 |
| `skin.risu.logo_width` | 로고 이미지 너비 (CSS 값) | `auto` |
| `skin.risu.brand_color` | 브랜드 색상 오버라이드 (`#RRGGBB`) | `#FFC0CB` |
| `skin.risu.footer_html` | 푸터에 표시할 HTML | 없음 |

## 사용자 설정
스킨 설정 모달에서 변경할 수 있습니다.
| 키 | 설명 | 기본값 |
|---|---|---|
| `risu.sidebar` | 사이드바 위치 (`right`/`hide`/`footer`) | 데스크톱 `right`, 모바일 `hide` |
| `risu.color` | 색상 프리셋 (`pink`/`lavender`/`mint`/`sky`/`peach`/`custom`) | 위키 설정 |
| `risu.color_custom` | `custom` 선택 시 사용할 `#RRGGBB` 색상 | 없음 |
| `risu.font` | 글꼴 (`pretendard`/`noto`/`custom`) | 시스템 기본 |
| `risu.font_custom` | `custom` 선택 시 사용할 기기 설치 글꼴 이름 | 없음 |
| `risu.floating_navbar` | 떠 있는 내비게이션 바 (스크롤 시 상단 유지) | `true` |
| `risu.reset_search_on_move` | 페이지 이동 시 검색창 초기화 | `true` |
| `risu.wide_mode` | 본문 최대 폭 확장 (1200px → 1600px) | `false` |
| `risu.admin_convenience` | [ADMIN] 관리 편의성 개선 | `true` |
